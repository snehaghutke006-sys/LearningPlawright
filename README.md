# Learning Playwright

Playwright examples and exercises.

Contents
- `Chapter1 Basics/` — simple example scripts.

Quick start

1. Install Node.js (LTS) and Git.
2. From the repository root run:

```powershell
npm install
# if you use Playwright tests:
npx playwright install
npx playwright test
```

Push to GitHub

```powershell
git remote set-url origin https://github.com/snehaghutke006-sys/LearningPlawright.git
git push -u origin main
```

Security note
- If you previously created a Personal Access Token and shared it, revoke it in GitHub Settings → Developer settings → Personal access tokens.
